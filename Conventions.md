# Naming Conventions

## Files names
 - Angular: dots
  -ex. main.controller.js

 - Everything else: Dashes
 - ex. bermuda-triangle.html


## Functions
 - camelCase

## Classes
  - Use Dashes

## IDs
  - Use camelCase

## HTML
  - double quotes ""

## JavaScript
  - single quotes ''

## Linter
  - JSlint
